# Insights

Dumping ground for raw thinking, half-formed ideas, observations, and research that don't yet belong in a structured playbook.

---

## What goes here

- **Random ideas** — content angles, offer experiments, partnership thoughts, naming brainstorms
- **Observations from the field** — "noticed competitor X is doing Y," "this prospect said Z verbatim"
- **Research notes** — links you saved, screenshots, articles worth referencing later
- **Experiment results** — "tried this outreach angle, got X response rate"
- **Customer quotes** — verbatim language from real conversations (gold for copy)
- **Anti-ideas** — things you tried that didn't work and why (so we don't repeat)

## What doesn't go here

- **Strategic decisions** → those belong in `.agents/playbooks/`
- **Positioning, voice, ICP** → that's `.agents/product-marketing-context.md`
- **Actual content drafts or scheduled posts** → those belong in `marketing/`
- **Code, configs, or deployment notes** → those belong in the project itself

## Format

One file per topic. Date-stamped at the top. Append-only — don't delete old thinking, even if you change your mind. Write `~~strikethrough~~` or add an "UPDATE:" note instead.

Examples of file names that could live here:
- `competitor-watch.md`
- `prospect-quotes.md`
- `pricing-experiments.md`
- `content-ideas-backlog.md`
- `partnership-targets.md`
- `2026-04-outbound-results.md`

## How to use it

When you (or Claude) start a new playbook or strategy doc, **read this folder first.** The best strategies come from accumulated raw observations, not first-principles brainstorming.
